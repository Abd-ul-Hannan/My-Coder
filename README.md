# My Coder

## Project Overview
My Coder is a sophisticated application designed to streamline coding activities and foster collaboration among developers. The project integrates various AI providers and Google Drive for enhanced productivity.

## Architecture
The system is built using a microservices architecture, where different components communicate through REST APIs. This approach enables easy scalability and maintainability.

## Google Drive Integration
My Coder leverages Google Drive APIs to allow users to save and share their projects seamlessly. Users can sync their code files with their Google Drive accounts, providing easy access from anywhere.

## AI Providers
The application integrates with multiple AI services, including:
- OpenAI
- Google AI
- Microsoft Azure AI

Users can choose the AI provider that best fits their needs and the project requirements.

## Quick Start Guide
1. Clone the repository:
   ```bash
   git clone https://github.com/Abd-ul-Hannan/My-Coder.git
   ```
2. Navigate to the project directory:
   ```bash
   cd My-Coder
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```

## Configuration
To configure the application, modify the `config.json` file:
- Set your Google Drive API credentials.
- Choose your preferred AI provider and configure its API settings.

## Troubleshooting
- Ensure all dependencies are installed correctly.
- Check API keys for Google Drive and AI providers if facing connectivity issues.
- Refer to the logs for any error messages that might guide you to the problem.
